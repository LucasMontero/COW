About

COW is a web application based on MEAN that allows developers to create and maintain their web pages directly on their hosting by modifying HTML, CSS and Javascript directly from the browser.

With this project, we want to offer an alternative to traditional CMS, that made dificult to enter directly into the code, facilitating the process of developing a static web page.
