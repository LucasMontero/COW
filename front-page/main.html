Web Page Panel Administrator

COW is a web application that allows developers to create webpages and maintain it via the web. It is an open source project and you can download and install it even for commercial use, learn more about the project in the About page.

Installation

COW is a administration panel for web pages based on MEAN(MongoDB, Express, Angular, Nodejs) that needs some packages for run. Perform the following steps for a successful installation:

  1. Make sure that your server has MongoDB, Nodejs and npm installed.
  2. Copy the "cow" folder on your server and install its depencies with "npm install".
  3. Set your server information to the "config.js" file.
  4. Open your web browser and run COW in the assigned port.

That's it! You can now use COW at your will! You can find a more detailed installation here and how to run cow in a nGinx server here.
